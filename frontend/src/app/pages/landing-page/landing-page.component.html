Welcome to the MMMKLCOBHT Webapp!

What do?

  1. Add a new Movie

    Navigate via the Menu-Bar to Add new Movie and paste an IMDB-Link in the input field.
    After submitting the Link with the button the result of the action appears below the input field.

  2. Get three random Movies from the List

    Navigate to Choose a Movie to get three random Movies from the list.
    Selecting a Movie with the Watch now button will copy it's name to the clipboard and mark it as watched (TODO) so it wont be suggested again.
    As long as the random Movies aren't refreshed its possible to mark a Movie as unwatched again in case we don't find a way to watch it.

  3. Check all Movies currently on the List

    Navigate to Check the List to get a detailed overview of all Movies currently on the list with the option to delete/update any Movie on it.
    Filter the list with the Show watched-Button to show/hide already watched movies.
